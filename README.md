# 🎮 Valto — 발로란트 내전 관리 디스코드 봇

발로란트 내전 모집, 참가자 관리, 랜덤 맵 추첨 기능을 제공하는 디스코드 봇입니다.

---

## 📁 프로젝트 구조

```
BOT/
├── Valto.py        # 메인 봇 코드
├── .env            # 환경변수 (깃허브 비공개)
├── .env.example    # 환경변수 예시
├── .gitignore
├── images/         # 맵 이미지 파일
│   ├── 스플릿.png
│   ├── 바인드.png
│   └── ...
└── logs/           # 로그 파일 (자동 생성/깃허브 비공개)
```

<br>

## ⚙️ 설치 및 실행

### 1. 패키지 설치

```bash
pip install discord.py python-dotenv
```

### 2. 환경변수 설정

`.env` 파일을 생성하고 토큰을 입력합니다.


```env
DISCORD_TOKEN=여기에_디스코드_봇_토큰_입력
```

### 3. 봇 실행

```bash
python Valto.py
```

<br>

## 🤖 명령어

| 명령어 | 설명 | 권한 |
|--------|------|------|
| `!명령어` | 명령어 목록 표시 | 전체 |
| `!내전 [날짜] [시간]` | 내전 모집 시작 | 전체 |
| `!맵` | 랜덤 맵 추첨 | 전체 |
| `!내전준비` | 참가자 소집 안내 | 전체 |
| `!내전마무리` | 내전 종료 안내 | 전체 |
| `!삭제번호 [번호]` | 특정 참가자/대기자 삭제 | 관리자 |
| `!순서바꾸기 [번호1] [번호2]` | 참가자 순서 변경 | 관리자 |
| `!내전초기화` | 참가자/대기자 전체 초기화 | 관리자 |
| `!종료` | 봇 종료 | 관리자 |

### 내전 참가 방법
명령어가 아닌 **이모지 리액션**으로 참가/취소합니다.
- 최대 **10명**까지 참가 가능
- 11번째 이후 신청자는 자동으로 **대기자 명단**에 등록

## 📋 사용 예시

```
!내전 2026-04-25 20:00
```
→ 해당 날짜/시간으로 내전 모집 공지 및 참가자 목록 생성

```
!맵
```
→ 랜덤으로 맵 하나를 추첨하여 이미지와 함께 표시
